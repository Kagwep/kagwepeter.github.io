Projects

Take a look at some of my recent projects:

Tenebo

A web app, "Tenebo", allows users to adopt wildlife animals as NFTs and use the proceeds to build shelters for affected communities. Adopters receive a unique digital certificate and regular updates on their adopted animal and the shelter projects. Our platform helps to solve human-wildlife conflict and support conservation and community development efforts.

Technologies used: Solidity, Hardhat, React JS

Live demo | Code repository

Joblinks

A platform that aggregates job postings from various sources and provides links to the original postings on the companies' websites. The goal is to make it easy for job seekers to discover and apply for job opportunities from a variety of companies in one place.

The platform also has an android application Joblinks

Technologies used: Python, Django, DRF, React Native

Live demo | Code repository

Hambre

"Hambre", aims to tackle hunger by connecting farmers with the resources they need to succeed. Through the platform, farmers can access financing and equipment, while users can rent idle land and purchase farm inputs. By providing these resources and connections, we hope to empower farmers to increase food production and alleviate hunger in their communities.

Technologies used: Near Protocol, Rust, React JS

Live demo | Code repository

Nyumbake

The aim of the above website is to help users find modern and spacious housing apartments in desirable locations, equipped with a variety of amenities to make their living experience comfortable and enjoyable. The website provides an easy-to-use platform for browsing and filtering available units, as well as viewing photos and virtual tours and learning more about the features and amenities offered. The website is designed to help users find their perfect housing apartment quickly and easily.

Technologies used: Python, Django, DRF, React JS
